mobile-first GitHub client that lets you manage code repositories efficiently anytime, anywhere.

## Core Features

**Repository Management**
After logging in, you can view your personal repository list. You can create new repositories, delete them, and modify their settings. Long-pressing a repository lets you quickly pin it, copy its link, download the source code archive, and view its branches and tags. Repositories support multiple sorting methods, such as by update time, star count, fork count, or name, making it easier to find projects.

**Star and Discovery**
A dedicated starred pages feature allows you to manage all your favorite projects and quickly unstar them. The home page also offers recommendatiohelping you discover high-quality open-source projects in the community.

**Global Search**
Supports searching for both repositories and users. Results are displayed in real-time as you type. Search history is automatically saved, making it easy to reuse common queries.

**Code Browsing**
When you enter a repository, you can view the complete file directory and preview code content online by clicking on files. Image previews are supported, and README files are automatically rendered as web pages, providing a reading experience close to that of the desktop.

**GitHub Pages Deployment**
Choose any of your repositories and enable GitHub Pages static website deployment with one click. You can monitor the deployment status in real-time. After deployment, you can directly access the website address, and you can also bind it to a custom domain name.

**Notification Center**
Receive GitHub notifications in one place so you won't miss important updates like Issue comments and Pull Request changes.

**Personal Homepage**
Displays data such as your contribution calendar, number of public repositories, and followers, giving you a full picture of your activity on GitHub.

## User Experience

**Interface Design**
Follows Material Design 3 design guidelines for a clean and clear interface with card-based layout for clear information hierarchy. Supports three theme modes: light, dark, and system, ensuring it's not harsh on the eyes when used at night.

**Interaction Details**
Comprehensive support for gestures like pull-to-refresh, swipe-to-return, and long-press to bring up the action menu. Page transitions have smooth animations, and operation feedback is immediate and clear.

**Multilingual Support**
Built-in Chinese and English languages, with free switching in settings.

## Security and Privacy
Uses OAuth official authorization for login without storing user passwords. All data is transmitted over HTTPS, and local caches can be manually cleared at any time.
